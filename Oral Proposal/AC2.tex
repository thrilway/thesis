      \pause
    \item Standard Minimalist syntax is ill-equipped to account for this
      \pause
      \begin{itemize}
	\item If a DP is licensed at some point in a derivation, it stays licensed
      \end{itemize}
      \pause
    \item In (modified) Label Theory, however \ldots  
  \end{itemize}
